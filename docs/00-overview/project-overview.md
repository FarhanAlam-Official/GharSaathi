## ğŸ“ Project Description

**GharSaathi** (literally meaning "Home Companion" in Nepali) is a full-stack web application designed to address the challenges in Nepal's property rental market. The platform serves as a centralized marketplace where:

- **Landlords** can list their properties, manage rental applications, and track lease agreements
- **Tenants** can search for properties, submit applications, and manage their rental experience
- **Administrators** can oversee platform operations, moderate content, and ensure quality
